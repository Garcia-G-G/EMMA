Licencia

Emma es software de uso personal y cerrado. Al instalar aceptas usarlo para tu propio uso. No se concede permiso para redistribuir, revender ni descompilar.

Privacidad (lo importante)

  • Tu memoria, tus secretos y tus archivos nunca salen de tu Mac: se guardan en ~/Library/Application Support/Emma, en ~/.emma y en el Llavero.
  • La voz se procesa con la API Realtime de OpenAI para que Emma te entienda y responda. Sin intermediarios, sin publicidad, sin telemetría.
  • Las llaves de API se guardan solo en el Llavero de macOS, nunca en texto plano.
  • Emma pide permisos (micrófono, automatización, accesibilidad) de forma explícita. Puedes revocarlos cuando quieras en Ajustes → Privacidad y seguridad.

Para desinstalar: ejecuta ~/Library/Application Support/Emma/installer/uninstall_macos.sh (usa --wipe para borrar también tu memoria local).
