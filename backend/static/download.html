Instala Emma

Un comando y en 3 minutos Emma vive en tu Mac. macOS 14+ (Sonoma) en Apple Silicon.

curl -fsSL https://theemmafamily.com/install.sh | sh

Qué hace el instalador:

1. Verifica tu Mac (macOS 14+ · Apple Silicon).

2. Baja uv (Python portable) — no necesita Homebrew.

3. Baja el código de Emma, instala deps, descarga el wake word model.

4. Pide permisos macOS (mic, accesibilidad, automatización).

5. Te dicta un código; mételo en theemmafamily.com/pair para vincular tu Mac.

6. Deja Emma corriendo. Di "Hey Emma" para hablarle.

Para revisar antes de correr: ver el script · código fuente · desinstalar: curl -fsSL https://theemmafamily.com/uninstall.sh | sh
